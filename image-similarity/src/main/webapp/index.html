Image search service reusing Assets IR-Image

IRImage

IRImage is the service, which allows to search visually similar images performing images content-based searching

Method Response Path Function
GET TEXT [/image-similarity/.]/component Display the name of the current component
GET XML/JSON [/image-similarity/.]/searchById.json?queryImageId=&start=&rows=&wskey=&profile= search images similar with an image available in the index using default similarity algorithm
GET XML/JSON [/image-similarity/.]/searchById.json?queryImageId=&start=&rows=&wskey=&profile= search images similar with an image available in the index using the Dominant Color based similarity algorithm
GET XML/JSON [/image-similarity/.]/searchByUrl.json?queryImageUrl=encodedUrl&start=&rows=&wskey=&profile= search images similar with an image available at the given Web location using the default similarity algorithm
GET XML/JSON [/image-similarity/.]/searchByUrl.json?queryImageUrl=encodedUrl&start=&rows=&wskey=&profile= search images similar with an image available at the given Web location using the Dominant Color based similarity algorithm
